# Modules
## Airflux Commons Text
This is a library of utilities for text.

### Installation

```kotlin
repositories {
    maven {
        url = URI("https://jitpack.io")
    }
}

dependencies {
  implementation("com.github.airflux:airflux-commons-text:Tag")
}
```
